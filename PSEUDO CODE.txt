
        QUIZ - 1M : 
           [Palindromic Prime]

	1. START
	2. Enter range
	3. Determine elements starting from '2' within the input range in array form using For - Loop
	4. Identify the prime numbers
	5. Get the binary of the prime and then reverse it
	6. Print "PALINDROMIC PRIME", if the binary is equal to the reverse 
	7. END



        QUIZ - 2M : 
            [Wasteful Number]

	1. START
	2. Enter numbers, then break if '0' is entered
	3. Determine the prime factors in each of numbers, then count the factors
	4. Count the number of digits of the specific number
	5. Compare the number of digits and prime factors
	6. If prime factors is less than digits, print "NON-WASTEFUL NUMBER"
	7. Else, print "WASTEFUL NUMBER"
	8. END



        MIDTERM : 
           [Palindromic Prime of the Wasteful Prime factors]

	1. START
	2. Input your desired range
	3. Determine elements starting from '2' within the range in array form using For -Loop Statement
	4. Determine prime factors of each element, then count
	5. Count the number of digits of the specific element/number
	6. Compare the number of digits and prime factors
	7. If factor is greater than the digits, print "WASTEFUL NUMBER"
	8. Get the binary of each of the wasteful prime factors, and then reverse it 
	9. If binary is equal to reverse, print all "PALINDROMIC PRIME FACTORS" of the specific wasteful number
	10. END
